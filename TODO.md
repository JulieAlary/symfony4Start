### Install Webpack

**Create folders**

~~~
./assets/js
./assets/css
~~~

**Create files**

~~~
./assets/js/app.js
./assets/css/app.css
~~~

**Add Webpack Encore**

~~~
yarn add @symfony/webpack-encore --dev
~~~

**Create the file**

~~~
./webpack.config.js
~~~

**When files above is create and often**
~~~
yarn run encore dev
~~~

**Install many dependencies**

~~~
npm install --save jquery
yarn add jquery --dev
yarn run encore dev
yarn add bootstrap
yarn run encore dev

~~~
